# Rain Pit Strategy AI - Frontend

Professional React frontend for the Rain Pit Strategy AI application.

## Features

- **Real-time Dashboard**: Live weather monitoring with professional charts
- **AI Strategy Analyzer**: Input weather conditions and get AI-powered recommendations
- **Pit Stop Calculator**: Optimize pit stop timing based on race conditions
- **Professional UI**: Racing-themed design with smooth animations

## Tech Stack

- React 18
- Vite
- React Router
- Tailwind CSS
- Recharts for data visualization
- Lucide React for icons
- Axios for API calls

## Installation

```bash
npm install
```

## Development

```bash
npm run dev
```

Frontend runs on http://localhost:3000

## Build for Production

```bash
npm run build
```

## API Configuration

The frontend connects to the backend API at `http://localhost:5000/api`

Configure in `src/api.js` if needed.
